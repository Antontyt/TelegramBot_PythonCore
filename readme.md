# 🤖 AI Telegram Bot

Многофункциональный Telegram-бот на базе OpenAI (ChatGPT + Whisper).
Отвечает на вопросы, ведёт диалог от лица известных личностей,
проводит викторину и распознаёт голосовые сообщения.

---

## ✨ Возможности

- 🎲 **Рандомный факт** — короткий научный факт по запросу
- 💬 **ChatGPT интерфейс** — вдумчивый ответ на любой вопрос (с анти-галлюцинационным промптом)
- 🎭 **Диалог с известной личностью** — общение от лица Илона Маска, Стива Джобса, Опры Уинфри
- 🧠 **Квиз** — викторина с генерацией тем и вопросов, проверка ответа по смыслу
- 🎤 **Голосовой ChatGPT** — распознавание речи через Whisper (русский язык)
- 📊 **Статистика** — учёт количества запросов к GPT

---

## 🛠 Технологии

- **Python 3.10+**
- **aiogram 3.x** — асинхронный фреймворк для Telegram Bot API
- **OpenAI API** — ChatGPT (gpt-4o / gpt-4o-mini) + Whisper
- **asyncio** — асинхронная обработка
